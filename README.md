# 行銷渠道成本效益分析

## 專案簡介
透過 **CPA** 與 **各行銷渠道花費與產品銷量的相關係數**，評估不同渠道的成本效益與未來潛力，並根據洞察提出預算配置建議

---

## 解決核心問題
**哪一個行銷渠道的花費最值得投入？**

---

## 分析指標

### 指標一：CPA
- **公式**：總花費 / 總銷量  
- **衡量**：渠道的成本效益

### 指標二：花費與銷量的相關係數
- **計算方式**：渠道花費 <> 銷量  
- **衡量**：渠道的未來潛力  

---

## 分析策略

| CPA 表現 | 相關係數 | 策略建議 |
|--------|--------|--------|
| 低 | 高 | 未來應增加預算 |
| 高 | 低 | 未來應減少預算 |
| 低 | 低 | 成本效益佳，但市場可能已飽和 |

## 洞察與建議

- **Affiliate Marketing**  
  - 平均 CPA 低、相關係數高
  - 最佳投資選擇，未來應增加預算  

- **Influencer Marketing**  
  - 平均 CPA 低、相關係數低  
  - 成本效益高，但市場可能已飽和  

- **TV**  
  - 平均 CPA 高、相關係數低  
  - 未來應減少預算
